# RestoPOS-Pro-Edition
Sistem Kasir (POS) Restoran berbasis Laravel. Fitur: Dashboard Laba Rugi Real-time, Sistem Poin Member, Pembayaran Multi-metode (QRIS/Tunai), Cetak Struk Thermal 58mm, &amp; Kitchen Display System. Solusi digitalisasi lengkap untuk manajemen operasional dan skalabilitas bisnis kuliner/UMKM.
